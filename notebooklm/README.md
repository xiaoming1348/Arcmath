# NotebookLM Source Pack

Use these files as the first-pass source pack for NotebookLM.

Recommended upload order:

1. `AMC8_Knowledge_Map.md`
2. `AMC10_Knowledge_Map.md`
3. `AMC12_Knowledge_Map.md`
4. `Placement_Report_Guide.md`
5. `../DIAGNOSTIC_BLUEPRINT.md`

Recommended NotebookLM outputs:

1. Student-facing "What does this exam test?" guide
2. Parent-facing "What does my child need to improve?" guide
3. Tutor-facing lesson planning summary
4. Placement report wording suggestions

Important rule:
- Keep the repository files as the system source of truth.
- Use NotebookLM to improve presentation, summaries, and explanation quality.
- Do not let NotebookLM become the authority for scoring, tagging, or diagnostic selection logic.
